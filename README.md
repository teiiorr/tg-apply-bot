## TG Apply Bot (Polling)

### Requirements
- Node.js 18+

### Setup
1) Create env:
   - cp .env.example .env
2) Fill:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_CHAT_ID
   - APP_TIMEZONE (optional, default: Asia/Tashkent)
   - LAB_SCREENPLAY_DEADLINE (optional, ISO date time)
   - LAB_VOICE_DEADLINE (optional, ISO date time)

### Run
npm start
